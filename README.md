# Installation

cinnamon and gnome

```git clone https://github.com/orangevictor/neolinux-wallpapers.git && cd neolinux-wallpapers && sudo cp -r xml/* /usr/share/gnome-background-properties/ && sudo cp -r -v *.png *.svg *.jpg /usr/share/backgrounds```
